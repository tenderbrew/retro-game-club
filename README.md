# Retro Game Club - Basic Website Layout

This project is a simple, responsive HTML & CSS layout you can customize for any small website.

## Files

- `index.html` - Main page structure (header, navigation, main content, sidebar, footer).
- `styles.css` - Styling and layout rules.

## How to view

1. Open the project folder: `C:\Users\ericm\Documents\retro_game_club`.
2. Double-click `index.html` to open it in your web browser.

You can edit the text, colors, and layout in `index.html` and `styles.css` to match your needs.
